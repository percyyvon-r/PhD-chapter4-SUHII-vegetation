# PhD Chapter 4 SUHII Vegetation

This repository contains the project scaffolding for Chapter 4 of the PhD work, focused on SUHII and vegetation analysis.

## Structure

- `data/` — input datasets and raw files.
- `notebook/` — Jupyter notebooks for exploratory analysis and results.
- `output/` — generated figures, tables, and export outputs.
- `scripts/` — reusable Python scripts and analysis utilities.

## Setup

1. Create and activate a virtual environment:

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
```

2. Install dependencies:

```powershell
python -m pip install -r requirements.txt
```

3. Run the sample entry point:

```powershell
python scripts\main.py
```
